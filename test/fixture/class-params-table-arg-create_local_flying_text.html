create_local_flying_text{text=…, position=…, create_at_cursor=…, color=…, time_to_live=…, speed=…}

Spawn a flying text that is only visible to this player.

Parameters
Table with the following fields:
  • text :: LocalisedString
  • position :: Position (optional)
  • create_at_cursor :: boolean (optional)
  • color :: Color (optional)
  • time_to_live :: uint (optional)
  • speed :: double (optional): Movement per second
Note: Local flying text is not saved, this means it will disappear after save/load.
Note: position or create_at_cursor is required.
Note: when create_at_cursor is true all values except 'text' are ignored.
